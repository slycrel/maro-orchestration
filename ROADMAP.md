# Roadmap

**This file is a stub by design (2026-07-04 docs refactor).** The phase-numbered
roadmap era ended around Phase 62; work is now tracked as:

- **MILESTONES.md** — the prioritized work queue (what to do next).
- **GOAL_BRAIN.md** — compiled truth, decisions, open threads. Wins on conflict.
- **BACKLOG.md** — active deferred items, bugs, ideas.
- **docs/history/ROADMAP_ARCHIVE.md** — every completed phase (0–62) with full
  detail, including the Phase 57–62 write-ups that used to live here.

North star (unchanged): give the system a top-level mission; it breaks it into
milestones, executes autonomously over days/weeks, learns from what works, and
reports progress without hand-holding. See `VISION.md` for the full intent and
`docs/ARCHITECTURE_OVERVIEW.md` for the Visibility → Reliability → Replayability
maturity doctrine and the visibility ladder (moved there from this file).

Do not add work items here — this file must stay free of `- [ ]` checkboxes so
the conversation miner treats MILESTONES/BACKLOG as the only queues.
